# School Management System

## Overview
This project is a more organized and efficient version of the previous School Management System. It aims to provide a user-friendly interface for all the actions and i am still working on  .

## Features
- Manage Instructors
  - View all instructors
  - Get instructor by ID
  - Get instructor by department ID
  - Get instructor by name
  - Add new instructor
  - Edit instructor details
  - Delete instructor

- Manage Courses
  - View all courses
  - Get course by ID
  - Add new course
  - Edit course details
  - Delete course

- Manage Departments
  - View all departments
  - Get department by ID
  - Add new department
  - Edit department details
  - Delete department
